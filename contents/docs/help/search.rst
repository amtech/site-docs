==============================
分类和搜索
==============================
.. Contents::




如何为文件进行分类？
==============================
- 易度系统提供标签功能，为文件打上对应标签进行分类；
- 可通过标签搜索到这一类文档


例如文件名(A1218-CCQ100B.xls)应该怎么搜索？
============================================== 
- 输入A1218、A12、A1218-CC、CCQ、CCQ100B、100B等都可搜索到


是否和百度、谷歌一样，对搜索词加亮？
============================================
- 易度搜索结果，支持对关键字加亮，能加亮提示搜索词在文本中的位置
