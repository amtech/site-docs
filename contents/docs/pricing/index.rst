上门服务
---------------
如果客户断网，或者需要更深入的实施咨询，希望更高效的完成实施，我们可提供上门服务(一般3天)。

上门前，需要预先进行必要的远程实施准备。具体实施内容和实施天数，需要根据具体情况而定。上门实施前，易度会提供详细的上门实施计划。上门实施，主要的上门实施内容可以包括：

1. 系统管理员系统管理指导
2. 文件结构规划
3. 权限分配
4. 数据导入
5. 结合客户日常工作，如何使用系统
6. 最终用户培训
7. 各种系统集成
